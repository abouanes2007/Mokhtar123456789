⚙️ دليل الإعداد والنشر

هذه الصفحة للأستاذ فقط — اتبع الخطوات لربط Firebase وEmailJS ونشر المنصة.

١. إعداد Firebase

1

إنشاء مشروع

اذهب إلى console.firebase.google.com ← Add Project ← أدخل اسم المنصة.

2

تفعيل Firestore

Build ← Firestore Database ← Create Database ← production mode

3

تفعيل Storage لرفع PDF

Build ← Storage ← Get Started — لرفع ملفات المحاضرات.

4

نسخ الإعدادات

Project Settings ← General ← Your Apps ← Web App ← انسخ firebaseConfig وضعها في كود الصفحة.

٢. إعداد EmailJS

1

إنشاء حساب مجاني

emailjs.com ← Sign Up — 200 بريد/شهر مجاناً.

2

ربط Gmail الجامعي

Email Services ← Add New Service ← Gmail ← أدخل بريدك الجامعي.

3

قالب بريد القبول

Templates ← Create: مرحباً to_name، تم قبولك. رابط المنصة: platform_url

4

نسخ المفاتيح

انسخ serviceId وtemplateId وpublicKey وضعها في الكود.

٣. النشر

1

رفع الملف على GitHub

أنشئ مستودعاً على github.com وارفع ملف HTML هذا مباشرةً.

2

الربط بـ Vercel أو Netlify

vercel.com أو netlify.com ← Import Project ← Deploy مجاناً.

3

رابطك الدائم

ستحصل على رابط مثل: sibawaih-platform.vercel.app

✅ بعد هذه الخطوات تصبح المنصة مباشرة بالكامل مع حفظ البيانات وإرسال البريد الحقيقي.
